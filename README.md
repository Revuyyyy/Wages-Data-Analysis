# Wage Dataset Analysis

## Project Overview

This project analyzes a wage dataset to understand the factors influencing worker wages. The analysis explores relationships between education, experience, union membership, and wages. It also includes statistical testing, predictive modeling, clustering analysis, and sampling strategy recommendations for further study.

The objectives of this analysis are to:

- Identify descriptive insights from the dataset
- Examine the distribution of wage values
- Detect and evaluate outliers
- Test hypotheses about wage differences
- Build predictive models
- Segment the dataset into homogeneous groups
- Recommend sampling strategies for future research
